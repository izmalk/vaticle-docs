== TypeDB

TypeDB is available in two editions: Cloud and Core.

[cols-1]
--
.xref:guides::installation/cloud.adoc[TypeDB Cloud]
[.clickable]
****
Install a production-ready and scalable version of TypeDB with advanced security and high-availability features.
****

.xref:guides::installation/core.adoc[TypeDB Core]
[.clickable]
****
Install a 100% open source version of TypeDB with everything developers need to start building on TypeDB.
****
--

== TypeDB clients

There are three types of clients available for TypeDB: Studio, Console, and language-specific drivers.

[cols-1]
--
.xref:guides::installation/studio.adoc[Studio]
[.clickable]
****
Install an IDE that facilitates the development process for TypeDB databases.
****

.xref:guides::installation/console.adoc[Console]
[.clickable]
****
Install a powerful CLI tool able to work in interactive(REPL) and non-interactive (script) modes.
****

.xref:guides::installation/drivers.adoc[Drivers]
[.clickable]
****
Install libraries that implement TypeDB protocol and provide a native API for programmatic access to TypeDB:
