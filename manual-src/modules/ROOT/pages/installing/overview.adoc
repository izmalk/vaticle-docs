== Deploy to cloud

Log in to https://cloud.typedb.com/[TypeDB Cloud] and create a new deployment on the Deployments page.

== Install TypeDB manually

TypeDB is available in two editions: Cloud and Core.

[cols-2]
--
.xref:manual::installing/cloud.adoc[TypeDB Cloud]
[.clickable]
****
Install a production-ready and scalable version of TypeDB with advanced security and high-availability features.
****

.xref:manual::installing/core.adoc[TypeDB Core]
[.clickable]
****
Install a 100% open source version of TypeDB with everything developers need to start building on TypeDB.
****
--

== TypeDB clients

There are three types of clients available for TypeDB: Studio, Console, and language-specific drivers.

[cols-1]
--
.xref:manual::installing/studio.adoc[Studio]
[.clickable]
****
Install an IDE that facilitates the development process for TypeDB databases.
****

.xref:manual::installing/console.adoc[Console]
[.clickable]
****
Install a powerful CLI tool able to work in interactive(REPL) and non-interactive (script) modes.
****

.xref:manual::installing/drivers.adoc[Drivers]
[.clickable]
****
Install a library that provides programmatic access to TypeDB with a native API in Rust, Python, Java, Node.js, and C++.
