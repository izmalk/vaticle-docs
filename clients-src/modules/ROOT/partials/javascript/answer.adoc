== Retrieve a mapping of variables to concepts

=== Syntax

[source,javascript]
----
conceptMap.map;
----

=== Description

Produces a Map where keys are variable names and values are [`concept`] s.

=== Returns

* `Map<String, Concept>`

== Retrieve concepts

=== Syntax

[source,javascript]
----
conceptMap.concepts();
----

=== Description

Return an iterator of all the [`concept`] s in this ConceptMap.

=== Returns

* Iterator of Concept
